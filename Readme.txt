The project consists two seprated applications: 
server-game application which run the server side of application
mini-game application which can be run on each client computer to start the mini-game

Both application contains config.properties file which specify Address and port of server machine. User
can change this information according to address of machine whih server will be run on it. By default, this information set to local host. 

Also, it is possible that user deifne number of gift boxes for mini-game. Although current mini-game is designed based on exactly for 12 boxes according to defenition of test project. But since the game logic of program is designed seperately it is proper to change the number of box to different ones.
